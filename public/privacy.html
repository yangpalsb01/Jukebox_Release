개인정보처리방침

최종 수정일: 2025년 5월

1. 수집하는 정보

JukeSync는 서비스 제공에 필요한 최소한의 정보만 처리합니다.

  • 방 이름, 호스트 닉네임 (사용자가 직접 입력)
  • 플레이리스트 및 곡 정보 (YouTube 영상 ID, 제목, 메모)
  • 공유 플레이리스트 코드 (24시간 후 자동 삭제)
  • 테마 설정, 볼륨, 사이드바 너비 (브라우저 localStorage에만 저장, 서버 전송 없음)

이메일, 전화번호, 실명 등 개인 식별 정보는 수집하지 않습니다.

2. YouTube API 서비스 사용

본 서비스는 YouTube API 서비스를 사용합니다. YouTube API 서비스를 통해 수집되는 데이터는 Google 개인정보처리방침의 적용을 받습니다.

본 서비스를 사용함으로써 사용자는 YouTube 서비스 약관의 적용을 받는 데 동의하는 것으로 간주됩니다.

3. 정보의 보관 및 삭제

방 정보는 서비스 운영 중 MongoDB에 저장됩니다. 방이 삭제되면 관련 데이터도 함께 삭제됩니다. 공유 플레이리스트 코드는 생성 후 24시간이 경과하면 자동으로 삭제됩니다.

4. 제3자 제공

수집된 정보는 서비스 운영 목적 외에 제3자에게 제공되지 않습니다.

5. 문의

개인정보 처리에 관한 문의는 본 서비스의 운영자에게 연락해주세요.

본 서비스는 YouTube API 서비스를 사용합니다. 사용 시 YouTube 서비스 약관이 적용됩니다.
